# root_checker

Plugin that allow flutter to detect rooted device. It's work in :

* Android
~~* IOS~~


## Getting Started

import the package
```phyton
    import 'package:root_checker/root_checker.dart'
```
call function isDeviceRooted 
```phyton
   Bool isDeviceRooted = await RootChecker.isDeviceRooted();
```
